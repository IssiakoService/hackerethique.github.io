Introduction à Bootstrap

Bootstrap est un framework front-end puissant et populaire utilisé pour créer des sites web et des applications web réactives. Voici un guide complet pour commencer à utiliser Bootstrap dans vos projets, avec de nombreux exemples pratiques.

1. Typographie

Bootstrap fournit des classes pour styliser les éléments de typographie tels que les titres, les paragraphes, les listes, etc.

Exemple de titres :

        
<h1>Titre 1</h1>
<h2>Titre 2</h2>
<h3>Titre 3</h3>
<h4>Titre 4</h4>
<h5>Titre 5</h5>
<h6>Titre 6</h6>

Exemple de paragraphes :

        
<p>Ceci est un paragraphe normal.</p>
<p class="lead">Ceci est un paragraphe de tête.</p>

Ceci est un paragraphe de tête.

Exemple de liste :

        
<ul>
  <li>Élément de liste 1</li>
  <li>Élément de liste 2</li>
  <li>Élément de liste 3</li>
</ul>

4. Cartes

Les cartes Bootstrap sont utilisées pour afficher du contenu dans des conteneurs stylisés avec des bordures et des ombres.

Exemple de carte simple :

        
<div class="card" style="width: 18rem;">
  <div class="card-body">
    <h5 class="card-title">Titre de la carte</h5>
    <p class="card-text">Ceci est du texte dans une carte Bootstrap.</p>
    <a href="#" class="btn btn-primary">Aller quelque part</a>
  </div>
</div>

5. Modals

Les modals Bootstrap sont des boîtes de dialogue superposées qui permettent d'afficher du contenu supplémentaire sans quitter la page actuelle.

Exemple de modal :

        
<!-- Bouton pour ouvrir le modal -->
<button type="button" class="btn btn-primary" data-toggle="modal" data-target="#exampleModal">
  Ouvrir le modal
</button>

<!-- Modal -->
<div class="modal fade" id="exampleModal" tabindex="-1" role="dialog" aria-labelledby="exampleModalLabel" aria-hidden="true">
  <div class="modal-dialog" role="document">
    <div class="modal-content">
      <div class="modal-header">
        <h5 class="modal-title" id="exampleModalLabel">Titre du modal</h5>
        <button type="button" class="close" data-dismiss="modal" aria-label="Close">
          <span aria-hidden="true">×</span>
        </button>
      </div>
      <div class="modal-body">
        Contenu du modal.
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-secondary" data-dismiss="modal">Fermer</button>
        <button type="button" class="btn btn-primary">Enregistrer les modifications</button>
      </div>
    </div>
  </div>
</div>
Titre du modal
Contenu du modal.

6. Formulaires

Les formulaires Bootstrap permettent de créer des formulaires stylisés et réactifs avec des classes CSS prêtes à l'emploi.

Exemple de formulaire :

        
<form>
  <div class="form-group">
    <label for="exampleInputEmail1">Adresse email</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp" placeholder="Entrez email">
  </div>
  <div class="form-group">
    <label for="exampleInputPassword1">Mot de passe</label>
    <input type="password" class="form-control" id="exampleInputPassword1" placeholder="Mot de passe">
  </div>
  <div class="form-check">
    <input type="checkbox" class="form-check-input" id="exampleCheck1">
    <label class="form-check-label" for="exampleCheck1">Se souvenir de moi</label>
  </div>
  <button type="submit" class="btn btn-primary">Envoyer</button>
</form>

7. Tableaux

Bootstrap permet de styliser les tableaux HTML avec des classes pour un affichage clair et réactif.

Exemple de tableau :

        
<table class="table">
  <thead>
    <tr>
      <th scope="col">#</th>
      <th scope="col">Prénom</th>
      <th scope="col">Nom</th>
      <th scope="col">Utilisateur</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">1</th>
      <td>Mark</td>
      <td>Otto</td>
      <td>@mdo</td>
    </tr>
    <tr>
      <th scope="row">2</th>
      <td>Jacob</td>
      <td>Thornton</td>
      <td>@fat</td>
    </tr>
    <tr>
      <th scope="row">3</th>
      <td colspan="2">Larry the Bird</td>
      <td>@twitter</td>
    </tr>
  </tbody>
</table>

8. Barres de navigation

Les barres de navigation Bootstrap sont utilisées pour créer des menus de navigation réactifs et stylisés.

Exemple de barre de navigation :

        
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#">Accueil <span class="sr-only">(current)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Fonctionnalités</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Tarifs</a>
      </li>
      <li class="nav-item">
        <a class="nav-link disabled" href="#">Désactivé</a>
      </li>
    </ul>
  </div>
</nav>
Navbar
  • Accueil (current)
  • Fonctionnalités
  • Tarifs
  • Désactivé

9. Grilles

Le système de grilles Bootstrap permet de créer des mises en page réactives et flexibles.

Exemple de grilles :

    <div class="col">Colonne 1</div>
    <div class="col">Colonne 2</div>
    <div class="col">Colonne 3</div>
  </div>
  <div class="row">
    <div class="col-6">Colonne 1 (largeur 6)</div>
    <div class="col-6">Colonne 2 (largeur 6)</div>
  </div>
</div>

10. Listes groupées

Les listes groupées Bootstrap sont utilisées pour afficher des listes d'éléments avec un style uniforme.

Exemple de liste groupée :

        
<ul class="list-group">
  <li class="list-group-item active" aria-current="true">Élément actif</li>
  <li class="list-group-item">Élément 1</li>
  <li class="list-group-item">Élément 2</li>
  <li class="list-group-item">Élément 3</li>
</ul>
